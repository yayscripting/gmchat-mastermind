Works via the GMChat (http://gmchat.nl).

Install guide:

1. Clone repo
2. Create directory named 'gamesecur'
3. chmod (0)777 'gamesecur'
5. Create following files:
	- bans545.txt
	- highscore29184.txt
6. chmod these files (0)777
7. have fun
