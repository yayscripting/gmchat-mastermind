Help (We raden je aan dit te lezen)
Welkom bij Mastermind, een leuk spel waarbij je de code van je tegenstander moet raden!

In het begin van het spel zie je een veld met 10 rijen met rondjes. De uitdager van het spel ziet 1 rij extra aan de rechter kant. Deze is aan gegeven met de knop start aan de onderkant. Als uitdager moet jij nu een code maken die je tegenstander moet proberen te raden. Klik op de cirkels (vanaf nu heten ze pinnetjes om een kleur te kiezen. Er zijn 6 kleuren: rood, geel, groen, blauw, roze en zwart. Zodra je een leuke en vooral lastige code denkt te hebben klik je op Start en je tegenstander mag beginnen te raden.

Op het scherm van de "rader" verschijnt nu een geopende (extra brede) kolom waarin jij je eerste zet mag raden. Het werkt het zelfde als het instellen van de code: Klik op de pinnetjes om ze van kleur te veranderen en klikt op Ok zodra je wilt kijken of deze code klopt.

Zodra je een code hebt verzonden zal je een nieuwe poging mogen doen aan de hand van een paar hints die je hebt gekregen. In het vakje waar zonet nog Ok stond staan nu 4 kleinere pinnetjes. Deze kunnen groen, rood of zwart zijn. De rondjes staan in willekeurige volgorde aangegeven en betekenen het volgende:
  • Groen: Een van de gekozen kleuren zit in de code van je tegestander op de goede plaats
  • Rood: Een van de gekozen kleuren zit in de code van je tegestander op de onjuiste plaats
  • Zwart: Een van de gekozen kleuren zit niet in de code van je tegestander
Aan de hand van deze informatie moet jij erachter te zien komen wat de code van je tegenstander is!

Voorbeeld 1
Simpele spel situatie die laat zie hoe ik aan de hand van groene en zwarte pinnetjes kan weten wat de code van mijn tegenstander is.



1. Ik gok 3 zwarte en 1 rood pinnetje, ik krijg 3 keer groen terug. Zwart en/of rood zit(ten) dus in de code van mijn tegenstander
2. Ik gok willekeurige kleuren om te kijken of deze er ook in voorkomen. (TIP!)
3. Ik herhaal stap 1 met een blauwe erbij om te kijken of ik nu een ander resultaat heb.
4. Ik weet door stap 2 dat geel niet in mijn code zit, ik kijk nu dus of rood er wel in zit. Ik krijg 4 zwarte terug dus rood zit er ook niet in.
5. Zwart moet er dus in zitten aan de hand van stap 1, ik kijk dit na.
6. Doormiddel van logisch nadenken gok ik er nu op dat alle 4 de pinnetjes zwart zijn en dit is correct!

Voorbeeld 2
Een situatie waarbij je aan de hand van rode, zwarte en groene pinnetjes moet beredeneren wat de code is.



1. Ik gok willekeurige kleuren om het spel goed te beginnen. Ik krijg 2 rode pinntjes terug dus dat betekend dat 2 van de 4 kleuren in de code van de tegenstander zitten maar op de verkeerde plaats.
2. Ik probeer of toevallig de rode en de blauwe de kleuren zijn die er inzitten. Ik krijg 4x zwart terug. Dit betekend dat geen van alle erin zit: geen rood, blauw of zwart! Dit is dus een hele handige zet.
3. De groene en de gele moeten er dan in zitten, ik weet dat ze niet op de goede plaats stonden dus wissel ik hun positie om. Dit klopt nog steeds niet dus moeten ze aan de buitenkant staan.
4. Ik probeer waar ik net achter ben gekomen en het blijkt te kloppen.
5. Ik heb nu alle 5 de kleuren gebruikt behalve roze, deze zal dus dan het middelste gat moeten opvullen en ik win het potje!
